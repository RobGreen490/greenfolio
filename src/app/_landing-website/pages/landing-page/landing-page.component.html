Employee Management System

A simple frontend solution for adding, updating, and deleting employees from a database. The current system does not persist data past a certain time frame. It does communicate with an ASP.NET backend client and features a responsive interface, real-time updates, and centralized routing for easy navigation between pages.

World Map

A world map interface that shows information about each country using an svg file and backend api from api.worldbank.org.
