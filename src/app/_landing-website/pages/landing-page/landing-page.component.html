Employee Management System

A simple frontend solution for adding, updating, deleting, and viewing employees from/to a database. data persist as the backend api communicates with an Azure SQL Database. This app features a responsive interface, real-time updates, and centralized routing for easy navigation between pages.

World Map

A world map interface that shows information about each country using an svg file and backend api from api.worldbank.org.
